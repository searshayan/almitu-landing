# almitu - Live Micro-Tutoring for English Learners

## About

almitu is a modern EdTech platform offering 25-minute live English tutoring sessions combined with AI-powered practice activities. We make language learning fit real life for busy professionals, newcomers, and anyone who wants to build English confidence.

## Setup & Deployment

### Local Development

Simply open `index.html` in a modern web browser.

### Production Deployment

1. **Update domain references**: Replace `https://www.almitu.com/` with your actual domain in:
   - index.html (canonical URL, Open Graph, Twitter Card)
   - sitemap.xml
   - .htaccess

2. **Add Favicon**: Create and add favicon files:
   - favicon.ico
   - apple-touch-icon.png (180x180)
   - favicon-32x32.png
   - favicon-16x16.png

3. **Add Social Media Images**:
   - Create `/images/og-image.jpg` (1200x630px for Open Graph)
   - Create `/images/twitter-card.jpg` (1200x600px for Twitter)

4. **Enable Analytics**: Uncomment the Google Analytics section in index.html and add your tracking ID

5. **SSL Certificate**: Ensure your hosting has a valid SSL certificate (required for HTTPS)

6. **Test**: Use tools like:
   - Google PageSpeed Insights
   - Google Search Console
   - W3C HTML Validator
   - Lighthouse (Chrome DevTools)
